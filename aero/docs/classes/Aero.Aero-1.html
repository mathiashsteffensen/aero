Static start

  • start(aero: typeof Aero): Promise<void>

  • Start the Aero application

    example

    The default start up script for an Aero application

    import Aero from "@aero/aero";

    Aero
      .initialize("config/Application")
      .then(Aero.start)
      .catch(Aero.logger.error)

    • aero: typeof Aero
