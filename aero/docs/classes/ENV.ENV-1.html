fetch

  • fetch(key: string, defaultValue?: string): string

  • Fetch a key from process.env

    remarks

    If no default value is provided and no value is found in the environment, an ENVError will be thrown because it is assumed your application requires these values

    example
      const maxDbPool = Aero.env.fetch("MAX_DB_POOL", "5")
      const minDbPool = Aero.env.fetch("MIN_DB_POOL", "1")

    Parameters

    • key: string

      the key to fetch from the environment

    • Optional defaultValue: string

      optional default value to return if none is found in process.env

    Returns string
